# 👨‍🏫PreMento
   **인공지능 발표 교정시스템입니다.**   
   **발표를 진행할 때 긴장감이나 두려움을 느끼는 사용자에게 도움이 되는 서비스입니다.**
  #### 🛠️ 제공 되는 기능은 다음과 같습니다.
   - 맞춤법 및 반복단어 검사  
   - 어색한 단어, 비속어 검사   
   - 음성 파일과 입력한 대본의 정확도 측정   
   - 입력한 파일의 수치화된 데이터 제공
